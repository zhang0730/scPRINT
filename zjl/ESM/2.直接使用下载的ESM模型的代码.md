```python
    cache_dir="/disk2/044/scprint/temp/data/esmc-600m-2024-12/data/weights"  # 指定本地路径

```

如果 `load_local_model` 方法不支持直接加载本地路径，你可以手动加载权重文件并初始化模型：

python

```python
import torch
from esm.models.esmc import ESMC

# 初始化模型
model = ESMC(d_model=1152, n_heads=18, ... )  # 根据模型定义填写参数

# 加载本地权重
state_dict = torch.load("/disk2/044/scprint/temp/data/esmc-600m-2024-12/data/weights/esmc_600m_2024_12_v0.pth", map_location="cuda" if cuda else "cpu")

# 加载权重到模型
model.load_state_dict(state_dict)

# 将模型移动到设备
model = model.to("cuda" if cuda else "cpu")
```
